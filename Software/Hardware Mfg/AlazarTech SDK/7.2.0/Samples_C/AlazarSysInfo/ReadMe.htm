Introduction

This folder contains a sample program that displays information about each AlazarTech digitizer board found in a computer. It demonstrates how to get a handle to a board, and use the handle to get information about the board.

 

This sample is part of the AlazarTech SDK, and is documented in the “AlazarTech SDK Programmer’s Guide”.

System requirements

Microsoft Visual C++ 6, Microsoft Visual C++ 2008 or later is required to build this C/C++ sample program. The sample includes a Visual C++ 6 workspace file for 32-bit platforms, and Visual C++ 2008 solution file for 32- and 64-bit platforms.

 

The AlazarTech version 5.7.7 or later drivers for your digitizer board are required to run this sample program. They can be downloaded from our web site.

http://www.alazartech.com/support/downloads.htm

Building the sample program

Microsoft Visual C++ 6

·        Open the “AlazarSysInfo.dsw” project workspace in the sample directory.

·        Select “Build | Set Active Configuration” menu command to display the “Select Active Project Configuration” dialog box, and select the “Debug” build or “Release” build as required.

·        Select the “Build | Build AlazarSysInfo” command to build the sample program.

 

If a build succeeds, it creates one of the following executables:

Debug

Debug\AlazarSysInfo.exe

Release

Release\AlazarSysInfo.exe

 

Note that the AlazarSysInfo is, by default, the directory name. For example, the output file of the debug build of the “\DualPort\TR_Header” sample is “\DualPort\TR_Header\Debug\TR_Header.exe”.

 

Microsoft Visual C++ 2008 Professional

·        Open the “AlazarSysInfo_x64.sln” solution in the sample directory.

·        Select the “Build | Configuration Manager” to display the “Configuration Manager” dialog and select the desired configuration and platform.

·        Select the “Build | Build Solution” command to build the sample program.

 

If a build succeeds, it creates one of the following executables:

32-bit Debug

Win32\Debug\AlazarSysInfo.exe

32-bit Release

Win32\Release\AlazarSysInfo.exe

64-bit Debug

x64\Debug\AlazarSysInfo.exe

64-bit Release

x64\Release\AlazarSysInfo.exe

 

Microsoft Visual C++ 2008 Express

·        Open the “AlazarSysInfo.sln” solution in the sample directory.

·        Select the “Build | Configuration Manager” to display the “Configuration Manager” dialog and select the desired configuration and platform.

·        Select the “Build | Build Solution” command to build the sample program.

 

If a build succeeds, it creates one of the following executables:

32-bit Debug

Win32\Debug\AlazarSysInfo.exe

32-bit Release

Win32\Release\AlazarSysInfo.exe
