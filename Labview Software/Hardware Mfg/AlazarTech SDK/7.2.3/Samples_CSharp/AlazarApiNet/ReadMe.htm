Introduction

AlazarTech supplies device drivers that allow applications to configure and acquire data from AlazarTech digitizer boards.

 

AlazarApi.cs and AlazarDSP.cs are C# source file containing a class that declares constants and functions exported by the AlazarTech device drivers so that they can be used by C# applications. It can be added as a source file to a C# application, or compiled into an assembly that can be referenced by the application.

 

This directory contains a C# solution that compiles the source files AlazarApi.cs and AlazarDSP.cs into the assembly AlazarApiNet.dll.

System requirements

Microsoft Visual C# 2008 or later is required to build the assembly from the source file.

 

The AlazarTech version 5.7.10 or later drivers for your digitizer board are required to run this program that uses AlazarApi.cs and AlazarDSP.cs as a source file or assembly. They can be downloaded from our web site.

http://www.alazartech.com/support/downloads.htm

Building the sample program

·        Open the “AlazarApiNet.sln” solution in the sample directory.

·        Select the “Build | Configuration Manager” to display the “Configuration Manager” dialog and select the desired configuration and platform.

·        Select the “Build | Build Solution” command to build the sample program.

 

If a build succeeds, it creates one of the following executables:

Debug

Bin\Debug\AlazarApiNet.dll

Release

Bin\Release\AlazarApiNet.dll
