Introduction

This utility program converts binary files to text files. It takes as its input a binary file containing two bytes per sample value, such as the sample values from an ATS310, ATS330, ATS460, ATS660, ATS9350, and ATS9462. And it outputs text with one with one sample value per line. This output can be redirected to a text file, and the text file can be opened in a text editor, spreadsheet programs, or other application that works with such files.

Using the sample program

To convert a binary file “data.bin” with one byte per sample to a text file “data.txt” with one sample per line, type the following at a command prompt:

 

      BinToText16 data.bin > data.txt

Building the sample program

Microsoft Visual C++ 6 or Microsoft Visual C++ 2008 is required to build this C/C++ sample program. The sample includes a Visual C++ 6 workspace file for 32-bit platforms, and Visual C++ 2008 solution file for 32- and 64-bit platforms.

 

Microsoft Visual C++ 6

·        Open the “BinToText16.dsw” project workspace in the sample directory.

·        Select “Build | Set Active Configuration” menu command to display the “Select Active Project Configuration” dialog box, and select the “Debug” build or “Release” build as required.

·        Select the “Build | Build BinToText16” command to build the sample program.

 

If a build succeeds, it creates one of the following executables:

Debug

Debug\BinToText16.exe

Release

Release\BinToText16.exe

 

Microsoft Visual C++ 2008 Professional

·        Open the “BinToText16_x64.sln” solution in the sample directory.

·        Select the “Build | Configuration Manager” to display the “Configuration Manager” dialog and select the desired configuration and platform.

·        Select the “Build | Build Solution” command to build the sample program.

 

If a build succeeds, it creates one of the following executables:

32-bit Debug

Win32\Debug\BinToText16.exe

32-bit Release

Win32\Release\BinToText16.exe

64-bit Debug

x64\Debug\BinToText16.exe

64-bit Release

x64\Release\BinToText16.exe

 

Microsoft Visual C++ 2008 Express

·        Open the “BinToText16.sln” solution in the sample directory.

·        Select the “Build | Configuration Manager” to display the “Configuration Manager” dialog and select the desired configuration and platform.

·        Select the “Build | Build Solution” command to build the sample program.

 

If a build succeeds, it creates one of the following executables:

32-bit Debug

Win32\Debug\BinToText16.exe

32-bit Release

Win32\Release\BinToText16.exe
