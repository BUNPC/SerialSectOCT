Introduction

Welcome to the AlazarTech SDK.

The ATS-SDK provides the files and documentation necessary to configure and acquire data from AlazarTech PCI/PCIe digitizer boards. It includes:

  • A programmer's guide
  • Header and library files
  • Sample Code

Getting started

A hardware user manual is available for each AlazarTech digitizer board model. It describes the features and specifications of the digitizer.

The AlazarTech SDK Guide is included with this SDK. It contains: a programmer's guide that describes how to configure and acquire data from AlazarTech digitizers; and an API reference that describes the functions exported by AlazarTech device drivers.

We recommend that you:

  • Read the hardware user manual to become familiar with the digitizer's capabilities.
  • Read the programmer's guide section of the ATS-SDK user manual to become familiar with the AlazarTech driver API.
  • Browse through the code samples. They provide examples that demonstrate how to configure and acquire data from your digitizer board.
