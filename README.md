# SerialSectOCT

This is the original version code for PSOCT system that uses the On-board FFT function of the Alazar card. It only outputs one channel and only the amplitude of the FFT. The data is saved bline by bline, the next version will be saving a whole cscan in one time.

This branch has been set to stall and will not be updated. Please refer to the master branch for the project.
